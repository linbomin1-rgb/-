# 腾讯云 COS + CDN 静态网站部署指南

本方案适用于**无需服务器**的场景，具有成本极低、访问速度极快（国内 CDN 加速）的优点。

## 1. 准备工作
*   下载我为您生成的 `salon_system_dist.zip` 并解压，得到 `dist` 文件夹。
*   登录 [腾讯云控制台](https://console.cloud.tencent.com/)。

## 2. 创建对象存储 (COS) 存储桶
1.  进入 **对象存储 (COS)** 控制台。
2.  点击 **存储桶列表** -> **创建存储桶**。
3.  **名称**：例如 `salon-system`。
4.  **地域**：选择离您最近的地域（如广州、上海）。
5.  **访问权限**：选择 **公有读私有写**（必须，否则外部无法访问）。
6.  完成创建。

## 3. 开启静态网站功能
1.  进入刚创建的存储桶，点击左侧菜单的 **基础配置** -> **静态网站**。
2.  点击 **编辑**，状态设为 **开启**。
3.  **索引文档**：填写 `index.html`。
4.  保存。此时您会获得一个“访问域名”，可以先测试是否能打开。

## 4. 上传文件
1.  进入存储桶的 **文件列表**。
2.  点击 **上传文件**，将 `dist` 文件夹内的**所有内容**（包括 `assets` 文件夹、`index.html` 等）直接拖入上传。
3.  **注意**：不要上传 `dist` 文件夹本身，而是上传它里面的内容。

## 5. 绑定您的腾讯云域名
1.  在存储桶菜单中，点击 **域名管理** -> **自定义源站域名**。
2.  点击 **添加域名**，输入您的域名（如 `salon.yourdomain.com`）。
3.  **开启 CDN 加速**：勾选此项，腾讯云会自动为您配置 CDN。
4.  **SSL 证书**：如果您有证书，可以在此处配置 HTTPS。
5.  按照提示在 **云解析 DNS** 中添加一条 CNAME 记录，指向腾讯云提供的目标地址。

---
**部署完成！** 您的系统现在可以通过您的自定义域名在国内公网极速访问了。
